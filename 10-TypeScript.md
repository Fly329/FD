# Typescript

### 安装

```javascript
npm install -g typescript
```

### 创建文件

**文件名.ts**

``` 
hello.ts
```

### 解析ts文件

**tsc 文件名.ts**

```
tsc hello.ts
```

### 运行

**node 文件名.js**

```
node hello.js
```

### 执行ts代码的两个步骤

``` 
1.tsc hello.ts
2.node hello.js
```

### 简化执行ts的步骤

```
1.安装命令：npm i-g ts-node
2.使用方式：ts-node hello.ts
```
